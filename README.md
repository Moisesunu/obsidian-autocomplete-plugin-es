# Autocomplete ES

Utilice la función de Autocomplete ES para escribir de manera más eficiente.

## Caracteristicas

- Un menú desplegable de palabras sugeridas de la palabra escrita actualmente.
- Reemplace la palabra escrita actualmente con una sugerencia del menú desplegable:
  - Seleccionar una sugerencia con el cursor
  - Seleccionar la sugerencia resaltada con las teclas `enter`/`tab`.
- Navegue por las sugerencias desplegables con las `up/down arrow` teclas.
- Ocultar el menú desplegable con la `esc` clave en cuyo punto todas las claves tienen sus funcionalidades predeterminadas
- Además, para utilizar la `enter` funcionalidad de tecla predeterminada, mantenga presionada adicionalmente la `shift` tecla: `shift + enter`

## Diccionario personalizado

El diccionario personalizado es una lista de palabras que seleccionas y que se añade al diccionario del plugin. Las palabras del diccionario personalizado se pueden sugerir en futuros menús desplegables. Puedes ver todas las palabras del diccionario personalizado desde la configuración del plugin.

#### Modificación
- Se pueden agregar palabras individualmente al diccionario personalizado de las siguientes maneras:
  - Desde la configuración del elemento
  - En el menú contextual del texto seleccionado (accesible mediante " clic derecho " )
- Las palabras se pueden eliminar individualmente del diccionario personalizado solo desde la configuración del elemento
- El diccionario personalizado también se puede restablecer en la configuración del complemento.

## Adicional

- El número maximo de sugerencias se puede modificar en la configuración del complemento (3-10)
- El complemento se puede desactivar desde la configuración del complemento
- El español es el único idioma admitido en este momento.
- Por el momento, todas las funciones están deshabilitadas dentro de los bloques de código (no se admite escribir dentro de los bloques de código)
